## ✨ Funcionalidades

- Cards com fundo branco, borda arredondada e sombra
- Efeito visual ao passar o mouse (aumento e sombra mais forte)
- Layout responsivo: os cards se ajustam conforme o tamanho da tela
- Cada card é um link que abre a pesquisa do filme no Google em uma nova aba

---

## 🎯 Objetivo da atividade

Atender aos seguintes requisitos:

- Estrutura com `header`, `main`, `cards` (mínimo 6) e `footer`
- Cada card possui título, imagem e texto
- CSS externo com fundo branco nos cards, borda, padding, margem, cor de fundo da página e fonte personalizada
- Uso de classes para estilizar os cards
- Cards posicionados lado a lado (com Flexbox)
- Efeito simples ao interagir (hover)

 
Tema: Velozes e Furiosos 🏁
